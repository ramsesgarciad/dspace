# Paquete de Despliegue - Repositorio IATECI 
 
Este paquete contiene todo lo necesario para desplegar el Repositorio IATECI. 
 
CONTENIDO: 
  - app/           : Aplicacion compilada 
  - scripts/       : Scripts de instalacion 
  - config/        : Archivos de configuracion 
  - GUIA_DESPLIEGUE_SERVIDOR.md : Guia completa 
 
INSTRUCCIONES RAPIDAS: 
 
1. Subir este archivo ZIP al servidor Ubuntu 
2. Descomprimir: unzip iateci-despliegue.zip 
3. Ir a la carpeta: cd iateci-despliegue 
4. Ejecutar: chmod +x scripts/desplegar-servidor.sh 
5. Ejecutar: sudo ./scripts/desplegar-servidor.sh 
 
Para mas detalles, consulta GUIA_DESPLIEGUE_SERVIDOR.md
